# 系统架构
> 环境 → 传感器 → 感知融合 → 决策规划 → 执行控制 → 车辆运动 → 环境
这是一个典型的闭环控制系统（Closed-loop system）
- perception
  - 作用：从传感器数据中识别环境信息。
  - 主要任务：
    - 目标检测
    - 车道线检测
    - 目标跟踪
    - 语义分割
    - 自车定位
- 融合
  - 作用：将多个传感器的信息进行融合提高感知精度和鲁棒性
  - 融合层次：
    - 数据级融合：直接融合原始数据
      - 例如：点云+图像  
    - 特征级融合：在神经网络特征层进行融合
    - 决策级融合：不同算法输出结果在进行融合
- 决策和规划
  负责决定车辆应该如何形式
  **分为** ：
  - 行为决策：决定车辆的驾驶行为
  - 轨迹规划：在确定行为后生成具体轨迹
- 执行控制
  决定车辆的驾驶行为：方向盘转角、油门、刹车
- 个人理解：
    - 自动驾驶的本质是：
    AI+*机器人系统*+*控制理论*+*系统工程*
    - 其中：
      - 感知：人工智能（深度学习）
      - 决策规划：AI+优化
      - 控制：经典控制/最优控制
      - 融合：概率估计
