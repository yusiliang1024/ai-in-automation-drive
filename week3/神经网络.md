# Neural Networks
1. Overview
   应用于：
   - 感知(Perception)
   - 特征提取（Feature Extraction）
   - 决策辅助（Decision Support）
   本质是学习一个mapping：
   >  $f(\theta)$ :Input → Output

2. From Pixels to Numerical Representation
   传感器数据（如图像）先被表示为数值形式：
    $$X\in \mathbb{R}^{H\times W\times C}$$
   - H:图像高度
   - W:图像宽度
   - C：通道数（RGB）
   本质：
   > 现实世界 → 像素 → 数值张量（tensor）
